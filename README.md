**Review System – HTML, CSS, JavaScript**
This project is a simple client-side review system where users can submit their username, comment, and rating, and the reviews are displayed on the right side of the page. 
All reviews are stored using localStorage, so they remain saved even after the browser is closed.

**Features**

Add review with:
- Username
- Comment
- Rating (1–5)

  Stores data in localStorage (no backend required)
  
Displays:
- Username
- Rating
- Comment
  
Time of submission

Automatically loads all previous reviews on page load

Clean two-column layout

Scrollable review box

**How It Works**

- User enters username, comment, and rating.
- Clicking Submit stores the review in the browser’s localStorage.
- All reviews are displayed in the review box on the right.
- Reviews remain saved even after reload since they are fetched from localStorage.

**Technologies Used**

- HTML – Structure
- CSS – Layout & styling
- JavaScript – Logic + localStorage

**How to Run**

- Download or copy the project files.
- Open index.html in any browser.
- Start adding reviews. They will appear instantly on the page.
